# Dialogue Project

## Overview

The Dialogue Project is a terminal command tool that allows users to wrap binaries with interactive prompts, making command-line interactions more user-friendly and efficient.

## Features
- Interactive command prompts for ease of use.
- Customizable binary wrapping.
- Supports various binaries and command-line tools.

## Installation

To install the Dialogue tool, clone this repository:

```bash
git clone https://github.com/bas080/dialogue.git
cd dialogue
```

## Usage

To use the Dialogue tool, run the following command:

```bash
./dialogue [binary_name] [options]
```

Replace `[binary_name]` with the name of the binary you wish to wrap.
